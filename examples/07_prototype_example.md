# Prototype Plan: LMS Career Pathway Builder

## User Flow Diagram
**Happy Path:**
1. User logs into LMS Home.
2. Clicks new "Pathway Builder" tab in the main navigation.
3. Completes a 10-card "Swipe Left/Right" interest assessment (e.g., "Working outdoors," "Solving complex equations").
4. Clicks "Generate My Map."
5. System displays 3 "Career Cards."
6. User clicks the top card to expand the "Market Opportunity" details.
7. User clicks "Pin this Pathway" to save it to their profile.

## Frontend (FE) Prototype Focus
The Figma prototype will focus heavily on **reducing cognitive load** and **visualizing data** beautifully.
*   **The Assessment UI:** Instead of a boring radio-button survey, we will prototype a mobile-friendly, Tinder-style swipe interface to gather explicit interest data quickly and playfully.
*   **The Recommendation Card UI:** This is the core screen. It must cleanly display:
    *   The Career/Major Title.
    *   A "Match Score" (e.g., 92% Match).
    *   An "Explainable AI" section: 3 bullet points explaining *why* (e.g., "Your 'A' in AP Physics," "Your interest in Design").
*   **Market Data Visualization:** An interactive, simple line graph showing projected job growth over the next 10 years, and a clear, large-font display of the median entry-level salary, utilizing green/red color coding for positive/negative market trends.
